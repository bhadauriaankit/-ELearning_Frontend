# E-Learning Examination Platform

A full-stack web application for online learning, test management, and secure examination workflows. The platform enables admins to create courses, modules, tests, and question banks, while students can attempt timed exams and receive instant results.

---

## 🚀 Features

### 👨‍💼 Admin / Instructor
- Secure login authentication
- Create and manage courses
- Add learning modules
- Create tests and question banks
- Assign marks and negative marking
- View student attempts and scores

### 🎓 Student
- Register and login securely
- Browse available courses/tests
- Attempt timed MCQ exams
- Instant score calculation
- View attempt history and results

### 🔐 Security
- JWT-based authentication
- Role-based authorization (Admin / Student)
- Protected APIs
